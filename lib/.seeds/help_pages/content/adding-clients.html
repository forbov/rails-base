Adding Clients

A client can perform the following:

  • To create tasks
  • To add carers
  • To view shift tasks

Clients can only be added by a Administrator. To add a client click the 'Add New Client' button from the Users screen.

The following fields are mandatory:

  • Email: Must be a valid email address and acts as a login so must be unique within the system
  • First and Last Name
  • Mobile Phone
  • Gender

Once the client is defined an invitation will be sent to them for them to join.
