Adding Managers

A manager can be appointed by a client to have all rights of the client:

  • To create tasks
  • To add carers
  • To view shift tasks

Managers can only be added by Clients. To add a manager click the 'Add Manager' button from the Managers tab on the Dashboard screen.

The following fields are mandatory:

  • Email: Must be a valid email address and acts as a login so must be unique within the system
  • First and Last Name
  • Mobile Phone
  • Gender

Once the manager is defined an invitation will be sent to them for them to join. They will be associated with the client who invited them.
